# 🚀 Start Your eCommerce Platform

## Quick Start

### Terminal 1 - Backend
```bash
cd backend
npm run dev
```
**Runs on:** http://localhost:5000

### Terminal 2 - Frontend
```bash
cd frontend
npm run dev
```
**Runs on:** http://localhost:3000
